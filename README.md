# InfiniTG

InfiniTG is an unlimited cloud storage and file management system powered by Telegram infrastructure.

Users can securely upload, organize, search, and access their files through a modern web interface. The system utilizes Telegram as the cloud storage backend while managing file metadata through a Laravel-based backend.

## Features

* User Authentication
* File Upload & Download
* Folder Management
* Search Files and Folders
* Image & Video Gallery
* Favorite Files
* Recent Files
* Recycle Bin

## Technologies

* Laravel (PHP)
* MySQL
* Tailwind CSS
* Python
* Telegram API (Telethon)
* Laragon
